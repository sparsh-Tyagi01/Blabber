## Getting Started

### 1. Clone the Repo
```bash
git clone https://github.com/sparsh-Tyagi01/blabber.git
cd blabber
npm install
```

### 2. Setup Environment
Create a `.env` file at the root:

```env
NEXT_PUBLIC_API_URL=http://localhost:8000
```

### 3. Run the App
```bash
npm run dev
```
